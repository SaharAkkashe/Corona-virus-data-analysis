# COVID-19 Data Analysis Dashboard

This project includes two data analysis dashboards: one built in **Excel** and the other in **Power BI**, both aimed at visualizing and analyzing COVID-19 data from various countries and regions. These dashboards offer insights into key metrics such as total cases, new cases, deaths, recoveries, active cases, testing rates, and vaccination statistics.

## Features
- **Excel Dashboard**: A detailed analysis using Excel with pivot tables, charts, and visualizations.
- **Power BI Dashboard**: An interactive dashboard built using Power BI to explore and filter COVID-19 data.
- **Data Insights**: Both dashboards provide visualizations for total cases, new cases, deaths, recoveries, active cases, testing rates, and vaccination percentages.
  
## Data Sources
The data used in this project is sourced from publicly available datasets regarding COVID-19 statistics globally. The data is regularly updated to ensure accuracy.

## Files Included
- **Excel Dashboard (`COVID19_Analysis.xlsx`)**: 
  - Contains pivot tables, graphs, and data visualizations.
  - Allows for detailed data analysis and filtering.
- **Power BI Dashboard (`COVID19_Analysis.pbix`)**: 
  - An interactive Power BI report for exploring COVID-19 trends across countries and regions.
  - Features slicers, charts, and graphs for in-depth analysis.
- **Data Files (`covid_data.csv`)**: 
  - The raw data used in both the Power BI and Excel dashboards.
  
## How to Use
1. **Excel**: 
   - Download and open the Excel file (`COVID19_Analysis.xlsx`).
   - Explore the data, use the pivot tables, and review the charts for insights.
   
2. **Power BI**: 
   - Open the Power BI file (`COVID19_Analysis.pbix`) using Microsoft Power BI Desktop.
   - Use the interactive filters and slicers to explore the data across different countries and regions.

## Technologies Used
- **Microsoft Excel**: For the Excel-based dashboard and data analysis.
- **Power BI**: For the interactive, visual dashboard.
- **CSV Files**: For raw COVID-19 data.

## Installation
- **Excel File**: No installation needed. Simply download and open the file in Excel or any compatible spreadsheet software.
- **Power BI File**: To use the Power BI dashboard, you need to have [Power BI Desktop](https://powerbi.microsoft.com/downloads/) installed. After installation, open the `.pbix` file to view the dashboard.
